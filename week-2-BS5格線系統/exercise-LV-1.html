Bootstrap grid examples

在 Bootstrap 網格系統中構建的基本網格佈局。

在範例中，.themed-grid-col 樣式加到主題中。預設的 Bootstrap 中可不使用。

Five grid tiers (五種格線層級)

Bootstrap 格線系統一共有五個層級，分別給不同的裝置尺寸使用。若未被其他層級覆蓋，每個層級會從一種最小的可視區域開始，並自動去適應其他更大的裝置。

Three equal columns (三個等距欄位)

取得三個等寬的欄 (columns)， 從桌面開始到大型桌面 。在移動設備、平板電腦及以下設備上，欄 (columns) 將自動成為通欄 (width: 100%)。

.col-md-4
.col-md-4
.col-md-4

Three equal columns alternative (由 .row 決定內欄的寬度)

用 .col 建立等距欄位，在上一層的 class .row 名稱在加 .row-cols-md-3（後方數字決定要幾個欄位）

.col child of .row-cols-md-3
.col child of .row-cols-md-3
.col child of .row-cols-md-3

Three unequal columns (三個不等距欄位)

從桌面開始取得三欄， 接著廷展到各種寬度的大型桌面裝置尺寸。 請記住，對於單個水平區塊，網格欄的總和應為 12。不僅如此，無論視窗大小如何，欄都成為通欄。

在行動裝置尺寸、平板電腦及以下尺寸時，這些欄及其嵌套欄將堆疊。

Mixed: mobile and desktop (混合：行動版和桌面版)

Bootstrap v4 網格系統有五層 class：xs (超小，這個 class 中綴不使用)、sm (小)、md (中)、lg (大) 和 xl (超大)。

每一層 class 都會向上擴展，如果您規劃為 md、lg 和 xl 設定相同的寬度，則只需指定 md。
